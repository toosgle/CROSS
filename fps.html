测试目标：Gallery组件

  • native 端: 基于 android.support.v4 包的 ViewPager 类进行开发
  • web 端:基于 Flipsnap 开发

测试方法

对 Gallery 进行频繁的切换操作，每秒钟对 fps 进行一次统计，当切换到最后一张图时，取得该轮测试的 fps 最大值 / 最小值 / 平均值

测试平台

  • android 4.0 及以上均使用 Chrome
  • android 4.0 以下，因为不支持 Chrome，均使用了系统自带浏览器
